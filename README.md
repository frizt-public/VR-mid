## Project Descrip
[This project is a relaxing and interactive VR experience set in a cozy bathroom environment. The core gameplay is simple yet highly satisfying: players can pick up various everyday objects and wash them clean using physics-based hand interactions. It is designed to be a lightweight, visually pleasing, and stress-free casual game that focuses on the tactile feel of VR.]
## Main goal
[The main goal of this project is to demonstrate core VR mechanics within a confined, detailed environment using Godot and OpenXR. Specifically, it showcases object grabbing, spatial manipulation, and dynamic state changes (e.g., transitioning an object from "dirty" to "clean") to create an immersive user experience.]
